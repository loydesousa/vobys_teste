# Teste Técnico Vobys
-- Nesse repositório segue o código feito para o teste técnico da Empresa Vobys, o código foi escrito em Pyhton no ambiente de código do Visual Studio Code e SQL no ambiente de código do MySQL Workbench.

-- O primeiro objetivo era conseguir puxar dados de uma api e em seguida transformar esses dados que recebemos em formato json para o formato de dicionários para facilitar o manuseio.

-- O segundo objetivo é guardar esses dados em um banco de dados, aqui utilizaremos o MySQL, e criarmos uma tabela a partir disso para realizarmos select's, insert's e delete's nesse banco de dados.

-- Por fim, concluirmos o teste fazendo o deploy desse projeto.
